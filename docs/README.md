# Music Composing DL

Music Composing DL is a streamlit app that generates **Bach Chorales**. It generates new chorales from a dataset of bach chorales. It was trained using convolutional neural networks which provies better feature extraction than any machine learning algorithm. The music generated will be available to listen and downloaded using an mp3 player.

## 📦 Technologies

- `Python`
- `Pandas`
- `Numpy`
- `Music21`
- `Pytorch`
- `Streamlit`
- `Convolutional Neural Network`

## 🎯 Features

Here's what you can do with Music Composing DL:

- Generate new music chorale.
- Listen to the Synthesized Bach Chorale generated by Convolutional Neural Networks.
- Download the generated Bach Chorale on the local machine.
- Review the chords generated by the CNN model.

## 👨‍🎓 The Process

This project began as a simple idea: "What if we could generate new Bach Chorales using deep learning?" I wanted to create an application that could learn from classical compositions and generate authentic new music. The journey started with understanding music theory and neural network architectures; I researched how CNNs could extract musical features and patterns from existing Bach Chorales.

Then came the technical challenges: preprocessing MIDI data for training, designing an effective CNN architecture for music generation, and building an intuitive Streamlit interface for users to interact with the model. The focus was on accessibility—allowing users to generate, listen to, and download synthesized Bach Chorales with just a few clicks.

## 🏗️ How I Built It

Building Music Composing DL was like conducting an orchestra; every piece needed to work in harmony:

### Phase 1: The Foundation
I began to plan the user requirements. I researched about Bach Chorales and how to use Convolutional Neural Nwtworks for generating synthetic Bach Chorales. Then I tried to think about what operations a user can perform on my web app and I mentioned them as functional depandencies.

### Phase 2: Data Preprocessing and Model Training
To generate the Bach Chorales, I needed a pre-existing dataset to get those naturally generated chorales. I cleaned and preprocessed the dataset according to most appropriate input to my convolutional Neural Network. Then, I used pytorch for training model and generating confusion matrix for the resulting model.

### Phase 3: The User Experience (Streamlit App)
Once the model was complete, I implemented a basic layout for Bach Chorale generating Streamlit app. I made a single page website that consists of a generate new music and then listen to that music.

### Phase 4: The Deployment
The final phase was about bringing Music Composing DL app to the internet. I deployed Music Composing DL on Streamlit so that the users can access mt project.

## 🪧 Running the Project

To run the project in your local environment, follow these steps:

1. Clone the repository to your local machine.
2. Run `cd music-composing-dl` and create a virtual environment to run the project
3. Run `pip install -r requirements.txt` to install the depandencies.
4. Run `streamlit run app.py` to open the project on your web browser
